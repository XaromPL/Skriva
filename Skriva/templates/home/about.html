About Skriva

What is Skriva? Skriva is a blogging app that allows you to create and manage your own personal blog. Technologies used

Django-Compressor

Combines JS and CSS files into one to reduce HTTP requests.

django-simple-captcha

Free CAPTCHA for login forms without API keys.

django-allauth

Secure login and registration with social logins support.

django-ckeditor

Rich text editor for blog content with styling options.

HTML / CSS / JS

Core frontend technologies for structure, style, and interactivity.
